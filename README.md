# 🍽️ Restaurant Management System

## 📌 Overview

The **Restaurant Management System** is a web-based application that allows users to register, browse restaurants, and order food online. Designed primarily for academic use, this project demonstrates core web development concepts using PHP, SQL, and front-end technologies.  

> ⚠️ *This project is not intended for commercial or production use.*

## 🚀 Features

- User registration and login  
- Browse restaurant menu and dishes  
- Place orders online  
- Admin panel for managing restaurants, dishes, and orders

## 🛠️ Technologies Used

- **Backend:** PHP, MySQL  
- **Frontend:** Bootstrap 4, jQuery, AJAX  
- **Database:** MySQL

## 🖥️ System Requirements

- XAMPP or similar LAMP/WAMP stack  
- Web browser

## ⚙️ Installation

1. Clone or download this repository to your local system.
2. Open **XAMPP Control Panel** and start **Apache** and **MySQL**.
3. Navigate to `http://localhost/phpmyadmin/` and create a new database.
4. Import the SQL file located in the `SQL/` directory into your new database.
5. Place the project folder in your XAMPP `htdocs` directory.
6. Open the project in a browser:  
   `http://localhost/your_project_folder/`

## 🔐 Admin Access

To access the admin dashboard, navigate to:  
`http://localhost/your_project_folder/admin/`

**Credentials:**
